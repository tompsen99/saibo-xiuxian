# 赛博修仙

复古文字聊天室武侠修仙MUD游戏

## 简介

2030年，你是一位60岁的退休老人，在一次深夜挂机中穿越到了武侠修仙世界。

这是一个90年代风格的纯文字MUD游戏，支持多人在线聊天、挂机修炼、门派系统、程序遗物收集等丰富玩法。

## 启动

```bash
npm install
npm start
```

然后打开 http://localhost:3000

## 技术栈

- 前端：HTML/CSS/JS（赛博风格终端UI）
- 后端：Node.js + Express + WebSocket
- 存储：JSON文件（MVP阶段）
- 部署：Cloudflare Workers + D1（计划中）
